# url_shortener
A URL Shortener web application built with Angular, Spring Boot, MySQL and Redis.

# tech stack
# frontend
HTML
Tailwind CSS
Angular
# backend
Spring
Spring Boot
Spring Data JPA
Hibernate
# storage
MySQL
# caching
Redis

# description
Built a web application using Angular for frontend, Spring Boot for backend, MySQL for database storage and Redis for caching to achieve better retrieval performance.
Added a intuitive rating (NPS) feature which stores the given rating in database.
Implemented pagination feature for better data handling on UI and faster performance.
